# 99_変更履歴

## Ver1.0

### 作成日

2026-07-07

### 内容

- Aurens HR Platform の基本構想を作成
- 既存勤怠管理システムを正マスターとする方針を確定
- 人事労務システムの初期対象機能を整理
- 入社手続き、従業員申請、Web給与明細、マイナンバー、e-Gov電子申請を対象機能に追加
- 給与計算は当面外部CSV取込とする方針を確定
- マイナンバーは初期版ではステータス管理のみとし、将来保存可能な設計にする方針を確定
- e-Govは最終的に送信まで対応する方針を確定
- 部門管理者の権限はユーザー設定で個別付与する方針を確定
