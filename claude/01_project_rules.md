# Claude Code Project Rules

## 1. Project

This repository is Aurens HR Platform.

The system is a cloud HR platform built with Next.js, TypeScript, Vercel and Supabase.

## 2. Read First

Before implementing any feature, read:

- README.md
- docs/13_development_standard_v1.4.md
- docs/14_development_charter_v1.4.md
- screens/
- database/
- api/
- prompts/

## 3. Do Not Build Everything At Once

Implement one issue or one screen at a time.

## 4. Source of Truth

Employee master data comes from the existing attendance system.  
Aurens HR stores only additional HR information.

## 5. Required For Each Feature

Each feature must include:

- UI
- API
- database access
- permission check
- validation
- error handling
- audit log where needed
- documentation update
